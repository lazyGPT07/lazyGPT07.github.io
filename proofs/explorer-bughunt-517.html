Finding 3: Direct Explorer Miner Route Returns 404

The explorer has a miner section on the dashboard, but /explorer/miners returns the server 404 page. If miner views are intended to be shareable/deep-linkable, the route needs SPA fallback or a deployed page. If not intended, the UI should avoid implying that direct route.
